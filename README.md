#BioREST

This package aim to retrieve data from biological database using REST api.

## Database available
* ArrayExpress
* BioDBnet
* BioGriD
* Biomart
* Encode
* Ensembl
* Eutils
* Fasta
* Intact
* KEGG
* Psicquic
* QuickGo
* Reactome
* String
* TogoWS
* Uniprot

## Requirement
* requests
* beautifulsoup4
* Pandas ( highly recommended )
